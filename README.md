# SCM Order & Inventory Visibility

This project focuses on analyzing supply chain operations using Power BI and Machine Learning.

## Features
- Order tracking dashboard
- Inventory monitoring
- Shipment delay analysis
- Demand forecasting using ML

## Tools Used
- Power BI
- Python (NumPy, Scikit-learn, Matplotlib)
- Excel

## Files Included
- .pbix → Dashboard
- .ipynb → ML model
- .xlsx → Dataset
- .pdf → Project Report
